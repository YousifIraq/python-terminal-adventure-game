# Python Terminal Adventure Game

A console-based adventure game developed in Python as part of my journey in learning software development and object-oriented programming.

This project focuses on designing a game using clean class structures, game logic, collision detection, and enemy behavior without relying on external game engines.

---

## Features

- Console-based gameplay
- Object-Oriented Programming (OOP)
- Multiple enemy AI
- Collision detection system
- Collectable key
- Life system
- Timer to track gameplay duration
- Multiple maps
- Win/Lose conditions
- Refactored architecture using separate classes

---

## Technologies Used

- Python 3
- NumPy
- Object-Oriented Programming
- Terminal/Console

## How to Run

1. Clone the repository

```
git clone https://github.com/YousifIraq/python-terminal-adventure-game.git
```

2. Navigate into the project

```
cd python-terminal-adventure-game
```

3. Install the required package

```
pip install -r requirements.txt

```

4. Run the game

```
python python-terminal-adventure-game.py
```

---

## Gameplay

The player must explore the map while avoiding enemies.

Objectives include:

- Navigate through the map
- Avoid enemy collisions
- Find the key
- Reach the exit
- Complete the game before losing all lives

Enemies move independently and create obstacles that require planning and timing.
